Das Kennwort wird als sicher genug angesehen, wenn es mindestens 10 Zeichen lang ist, mindestens eine Ziffer enthält und einen Groß- und einen Kleinbuchstaben enthält. Das Kennwort enthält nur lateinische ASCII-Buchstaben oder Ziffern.

Eingabe: Ein Kennwort als Zeichenkette (str{% if is_js %}ing{% endif %}).
