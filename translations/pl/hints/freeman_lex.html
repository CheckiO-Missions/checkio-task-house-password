Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę zadeklarować flagi logiczne dla dużych i małych liter oraz cyfry. Iteruj po podanym ciągu i zmień odpowiednią flagę na True jeśli pojawi się odpowiedni znak. Jeśli wszystkie flagi mają wartość True, funkcja musi zwrócić True, w przeciwnym razie - False.

Ok, od czego powinienem zacząć?

Zgodnie z opisem, hasło musi być dłuższe niż 9 znaków. Proszę to sprawdzić za pomocą if i dalej sprawdzać dłuższe ciągi znaków. Dla innych może Pan od razu zwrócić False. 

if len(data)> 9:
    ...

return False
Co powinienem zrobić dalej?

Proszę utworzyć flagi logiczne jako zmienne u dla dużych liter, l dla małych liter i d dla cyfr z wartościami początkowymi False, ponieważ nie sprawdza Pan jeszcze żadnych znaków. 

u = l = d = False

Teraz proszę wykonać iterację po podanym ciągu znaków za pomocą pętli for. 

for char in data:
    ...
Proszę mi pomóc sprawdzić każdy znak.

Proszę obsłużyć wszystkie sprawdzane przypadki za pomocą instrukcji if/elif. Proszę najpierw sprawdzić wielkie litery. Ponieważ jedna taka litera w haśle jest wystarczająca, należy sprawdzić znaki dla tego przypadku tylko wtedy, gdy wartość u jest nadal False. Teraz proszę sprawdzić, czy bieżąca wartość char jest wielką literą za pomocą metody str.isupper(). 

if not u and char.isupper():
    ...
elif ...:
    ...
elif ...:
    ...

W tym przypadku wystarczy zaktualizować wartość u za pomocą True, ponieważ znaleziono wielką literę w podanym ciągu znaków. Jeśli pełne wyrażenie jest wystarczająco krótkie, można zapisać je w jednym wierszu. 

if not u and char.isupper(): u = True
Jak sprawdzić małe litery?

Ramka jest taka sama, ale zamiast tego należy użyć metody str.islower(). 

elif not l and char.islower(): l = True

To samo dotyczy sprawdzania, czy znak jest cyfrą. 

elif not d and char.isdigit(): d = True
To jasne. Jak sprawdzić, czy całe hasło jest bezpieczne?

Po przetworzeniu znaków należy sprawdzić, czy wszystkie flagi mają już wartości True. Jeśli tak, nie ma potrzeby sprawdzania kolejnych znaków - hasło jest bezpieczne. 

if u and l and d: return True
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def checkio(data: str) -> bool:

    if len(data)> 9:
        u = l = d = False
        for char in data:
            if not u and char.isupper(): u = True
            elif not l and char.islower(): l = True
            elif not d and char.isdigit(): d = True
            if u and l and d: return True

    return False

Jasne, proszę łapać!

Za pomocą modułu wyrażeń regularnych sprawdzamy każdy wzorzec pod kątem obecności w podanym ciągu znaków. Te wartości logiczne są używane jako argument funkcji all(). 

import re


def checkio(d: str) -> bool:

    return len(d)> 9 and all(re.search(p, d) for p in ('[A-Z]', '\d', '[a-z]'))

Proszę przekonwertować ciąg na małe i wielkie litery i sprawdzić, czy ciąg pozostaje taki sam - jeśli tak, to nie ma w nim (odpowiedniej) małej i wielkiej litery. Na koniec proszę sprawdzić, czy w ciągu występuje jakakolwiek cyfra. 

def checkio(data: str) -> bool:

    if len(data) < 10:
        return False
    if data.upper() == data:
        return False
    if data.lower() == data:
        return False
        
    return any(c.isdigit() for c in data)
