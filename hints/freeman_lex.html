After a character processing you need to check if all your flags already have values True. If so, no need to check next characters - a password is safe.

So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def checkio(data: str) -> bool:

    if len(data) > 9:
        u = l = d = False
        for char in data:
            if not u and char.isupper(): u = True
            elif not l and char.islower(): l = True
            elif not d and char.isdigit(): d = True
            if u and l and d: return True

    return False

Using regular expression module check each pattern for being present in the give string. These logic values are used as argument of all() function. 

import re


def checkio(d: str) -> bool:

    return len(d) > 9 and all(re.search(p, d) for p in ('[A-Z]', '\d', '[a-z]'))

Convert the string into upper case and lower case and check if the string remains the same - if so, there no (respective) lower case letter and upper case letter in it. Finally, check if there is any digit in the string. 

def checkio(data: str) -> bool:

    if len(data) < 10:
        return False
    if data.upper() == data:
        return False
    if data.lower() == data:
        return False
        
    return any(c.isdigit() for c in data)
